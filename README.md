# AI 应用落地 Vault

这是一个面向 AI 应用落地的 Obsidian vault。目标是把全网优质教程、案例、大 V 经验、工具比较和实操技巧，编译成能直接使用的中文知识库和交付产物。

## 核心流程

```text
Clippings / Raw
  ↓
Wiki
  ↓
Outputs
```

- `Clippings/`：浏览器剪藏的 Markdown 文章和 blog，保留原样。
- `Raw/`：PDF、视频、截图、repo、数据等原始材料。
- `Wiki/`：AI 编译后的应用知识层。
- `Outputs/`：教程、案例拆解、工具比较、SOP、课程、PPT、图文卡片。
