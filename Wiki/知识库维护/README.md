# 知识库维护

这个主题维护本 vault 的知识生产方法：如何从 `Raw / Clippings` 进入 Wiki，如何合并旧页，如何防止腐烂，以及如何从 Wiki 生成 Outputs。

它不是正式 SOP 货架。可直接交付给用户的采编模板、维护检查清单和课程材料放在 `Outputs/`。

## 当前主线

- [[Wiki持续进化维护闭环|Wiki持续进化维护闭环]]：Wiki 合并优先、持续演进、防腐和 Output 回流规则。
- [[教程筛选到NotebookLM审片再到Wiki编译闭环|教程筛选到NotebookLM审片再到Wiki编译闭环]]：YouTube / NotebookLM / CourseKit / Clippings 的采编流程。

## 合并原则

- 涉及本 vault 结构、目录、合并、降级、Output 回流的规则，优先并入 `Wiki持续进化维护闭环`。
- 涉及教程搜索、审片、原始材料入库、CourseKit 的流程，优先并入 `教程筛选到NotebookLM审片再到Wiki编译闭环`。
- 如果某条规则已经可以直接交付给用户使用，再从本主题生成 `Outputs/SOP与模板/`。
