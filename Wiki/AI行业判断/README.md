# AI行业判断

高可信行业人物、公司、研究者对 AI 应用、软件范式、agent、商业机会和能力边界的前沿判断。

## 维护原则

- 行业判断页不追求当天新闻速度，重点沉淀能影响长期目录、课程和工具判断的观点。
- 单个大牛观点只作为框架，不直接当作事实；落到教程或 SOP 前要有可验证步骤或本库案例。
- 开源生态材料要记录作者、license、repo、商业关系和证据级别，避免只看热度和转发量。
