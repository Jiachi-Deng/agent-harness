# AI视频制作

用 AI 做视频选题、脚本、分镜、字幕、B-roll、动态视觉、缩略图和发布流水线。

这个主题先从真实教程和创作者工作流里自然生长。优先收录能直接复刻的内容生产方法，而不是泛泛讨论视频生成模型。

维护原则：先让现有主线吸收新材料。工具模块先并入对应生产任务，等后续积累出足够官方资料、工具比较、价格和实测案例后，再考虑重新拆成独立工具页。

## 五条主线分工

| 主线 | 负责什么 | 不负责什么 |
| --- | --- | --- |
| YouTube研究与脚本模仿 | 从 transcript 学表达结构、hook、脚本节奏和解释方式 | 不负责最终剪辑，也不负责短剧分集 |
| 讲解图与视觉画布工作流 | 把概念、脚本、课程结构或产品卖点变成可讲、可改、可导出的视觉画布 | 不负责时间轴、动效和 render |
| AI视频剪辑与动态图形工作流 | 粗剪、字幕、motion beats、overlay、转场、render 和最终发布版 | 不负责故事改编，也不负责本地模型节点排错 |
| ComfyUI本地视频生成入门 | 本地模型、节点、workflow、角色图、场景图、关键帧、短镜头、放大和排错 | 不负责剧情结构、分集节奏和最终剪辑 |
| AI短剧与漫剧制作流水线 | 故事改编、分集、剧本、连续性、角色 / 场景资产规格、分镜和镜头提示词 | 不负责具体模型安装，也不负责时间轴剪辑 |

## 材料归位规则

| 新材料主要讲什么 | 优先进入哪里 |
| --- | --- |
| 目标创作者分析、视频字幕、脚本模仿、hook 和讲法 | `YouTube研究与脚本模仿` |
| Excalidraw、Paper、视觉画布、课程图、缩略图、landing page 视觉稿 | `讲解图与视觉画布工作流` |
| 口播粗剪、CapCut、Remotion、Hyperframes、overlay、motion beats、render | `AI视频剪辑与动态图形工作流` |
| ComfyUI、模型目录、custom nodes、LTX、Flux、workflow JSON、upscale、显存排错 | `ComfyUI本地视频生成入门` |
| 小说改短剧、分集、剧本、角色连续性、分镜、首帧 / 尾帧、镜头提示词 | `AI短剧与漫剧制作流水线` |

如果一个材料同时覆盖多条线，不拆原文；把判断拆进对应主页面。比如短剧教程里出现 ComfyUI：剧情和分镜进入短剧页，节点和模型排错进入 ComfyUI 页，最终剪辑和动效进入剪辑页。
