# AI营销自动化

这个主题沉淀一人公司、小团队和创作者如何用 AI 组织研究、选题、素材、发布、商务邮件和日程协作。

它不是正式案例拆解页。正式案例、workflow 和运营 SOP 放在 `Outputs/`；这里维护背后的模块关系、输入输出、风险、权限边界和可复用判断。

## 当前主线

- [[AI一人营销团队|AI一人营销团队]]：当前主干页，说明研究、选题、视觉、媒体、商务和分发如何串起来；Gen Media、Buffer Publisher、Email Manager 三个 seed 模块已并入这页维护。

## 合并原则

- 讲整体营销系统、人机分工和模块关系的材料，优先并入 `AI一人营销团队`。
- 讲媒体素材工作台、idea queue、商务邮件筛选、日程建议和人工审批边界的材料，先并入 `AI一人营销团队` 的对应模块；只有材料持续膨胀并能独立支撑 SOP / 工具比较 / 权限台账时，再拆出独立页。
- 讲视频脚本、讲解图、剪辑、动态图形的材料，优先回到 [[Wiki/AI视频制作/README|AI视频制作]]。
- 讲个人收藏、剪藏、Readwise、资料回查的材料，优先回到 [[Wiki/AI知识库/README|AI知识库]]。
- 只有当材料能补强营销自动化的模块设计、权限边界、交付风险或业务流程时，才进入本主题。
